### 欢迎使用Resona Bot 

Resona Bot 是主要为PJSK繁体中文服务器设计的聊天机器人，也支持国服和日服

---

## 功能列表

### 1. **绑定账号 (tw/cn/jp绑定 默认隐藏id)**
- 绑定账号并设置为默认使用账号，支持多服多账号和单服多账号绑定，每个账号有独立编号
- 得益于引入了默认服务器和默认账号机制，指令均可以不指定服务器的情况下使用默认服务器

```
[服务器]绑定 游戏id
```
注：可以发送 `[服务器]给看` 打开其他指令的id显示

### 2. **查询个人信息 (个人信息/pjskprofile 允许@查询 在白名单群聊中允许指定位次或id查询)**

```
[服务器]个人信息 或 [服务器]pjskprofile
```

### 3. **查询排名 (sk 允许@查询)**
- 按指定的位次查询信息。不输入默认查询用户自己的排名信息。返回结果包含时速、分数等信息，最大查询6个位次

```
[服务器]sk [位次]
```

### 4. **查询单榜排名 (wl + 角色罗马音 允许@查询)**
- 查询单人排行榜信息。角色罗马音如 `mfy`、`knd` 等。返回结果包含时速、分数等信息，最大查询6个位次

```
[服务器]wl [位次] [角色罗马音]
```

### 5. **查询排名时速/ss/榜线/ptl (榜线/ptl/时速/ss)**
- 查询当前排名（如 100 名、200 名等）的时速或者分数线，添加wl参数可以查询worldlink期间的数据

```
[服务器]榜线/ptl/时速/ss 或 [服务器]wl榜线/ptl/时速/ss [角色罗马音]
```

### 6. **周回增量统计 (ptd)**
-返回玩家过去 xdxhxm 的周回增量数据图表和停车时间，默认为60分钟，输入f查询全量数据

```
[服务器]ptd [xdxhxm]
```
注：时间间隔小于6小时标记五分钟以上停车，小于12小时标记15分钟，小于1天标注30分钟，大于一天只标注停车大于1小时的时间段

### 7. **计算追杀时速 (追杀/zs)**
- 计算到活动结束, 指定排名2到指定排名1所需的最低时速 (未指定排名默认t100)，添加wl参数可以在worldlink期间使用

```
[服务器]zs[目标排名1] [目标排名2] 或 [服务器]wlzs[目标排名1] [目标排名2]
```

### 8. **查询停车信息 (csb 允许@查询)**
- 按指定的位次/角色查询停车信息。不输入位次默认查询用户自己。返回结果包含当前信息、分数，停车时间段详情，添加wl参数可以在worldlink期间使用

```
[服务器]csb/查水表 [位次] 或 [服务器]wlcsb/wl查水表 [位次] [角色罗马音]
```

### 9. **更新背景 (背景更新/背景更新 + 图片)**
- 发送带有图片的消息或回复一张图来更新个人信息背景。支持 `.jpg`/`.jpeg` 和 `.png` 格式

```
更新背景[图片] or 回复一张图片
```

### 10. **倍率计算 (倍率)**
- 计算倍率/实效，卡组技能请使用百分数 如：倍率160 150 150 150 150

```
倍率卡1技能 卡2技能 卡3技能 卡4技能 卡5技能
```

### 11. **活动pt计算 (活动点数)**
- 计算卡组在特定歌曲下的活动pt

```
活动点数 ——> 综合力 活动加成 倍率1 倍率2 倍率3 倍率4 倍率5 s6/返场倍率
```
注：虾默认10火Ex 龙默认5火Hard Sage默认5火Ex

### 12. **烤森资源 (msr)**
-返回烤森资源总结和地图透视，输入f强制查询过期的数据

```
[服务器]msr [f]
```

### 13. **活动组卡**

```
[服务器] 活动组卡 [活动id] [角色] [歌曲] [难度] [live类型] 实效[队友实效] [全当期（满破满技能满画布）] [1星/2星/3星/4星/生日(满破满技能满画布)] #[指定角色] c[强制包含的卡面ID（满破满技能满画布）]
```
- **服务器**：可选参数，支持 `jp` / `cn` / `tw`，不填使用默认服务器
- **活动id**：可选参数，需要组的活动 ID,不填则默认当前活动  
- **角色**：仅当前活动/指定活动id为 `worldlink` 时必填(例如：`mfy`)
- **歌曲**：可选参数，支持日文原名或部分 Haruki 数据库常用别名(如 “虾”、“火花”、“人生”)  
- **难度**：可选参数，支持缩写：  
  - ez, 绿 = easy
  - nm, 蓝, n = normal
  - hd, 黄, h = normal
  - ex, 红, e = expert
  - ma, 紫, m = master
  - apd, 粉/黑/白/彩, a = append
- **live类型**：支持 `solo` / `multi` / `auto`，默认为多人模式 
- **队友实效**：可选参数，例如268（等同3.68倍率），默认200 请不要填入不合理数据
- **强制包含的卡面ID**：可选参数，例如c765满破 c637满技能 表示765满破，637满技能
- **全当期**：可选参数，会假设你全当期进行组卡，当期数量大于 5 无法使用
- **指定角色**：可选参数，可跟多个角色，第一个作为队长（无法和指定卡面/全当期同时使用）

**示例：**
```
jp活动组卡 112 mfy 虾 ma auto
活动组卡 134 虾 ex solo
cn活动组卡 165 群青 apd 实效210 c195满破满技能满画布
tw活动组卡 180 虚无小姐 master 实效240 四星满破 c765满技能
```

### 14. **挑战组卡**

```
[服务器] 挑战组卡 [角色] [歌曲] [难度] c[强制包含的卡面ID（满破满技能满画布）]
```
- **服务器**：可选参数，支持 `jp` / `cn` / `tw`，不填使用默认服务器
- **角色**：指定挑战角色（例如：`mfy` / `knd`）  
- **歌曲**：同 “活动组卡”  
- **难度**：同 “活动组卡”  
- **强制包含的卡面ID**：同 “活动组卡”  

**示例：**
```
挑战组卡 mfy 群青 apd
```
注：不提供一键计算全部角色挑战组卡功能

### 15. **加成组卡**

```
[服务器] 加成组卡 [指定加成1] [指定加成2/3/4] c[强制包含的卡面ID（满破满技能满画布）]
```
- **服务器**：可选参数，支持 `jp` / `cn` / `tw`，不填使用默认服务器
- **活动id**：可选参数，需要组的活动 ID,不填则默认当前活动 
- **指定加成**：要求输出卡组的加成，允许输入多个，以空格分隔
- **强制包含的卡面ID**：同 “活动组卡”  

**示例：**
```
加成组卡 100 c[强制包含的卡面ID]
cn加成组卡 100 200 e112 mfy
```
注：为了方便区分 如果希望指定活动id需要加上e来识别

- **服务器**：可选参数，支持 `jp` / `cn` / `tw`，不填使用默认服务器
- **活动id**：可选参数，需要组的活动 ID,不填则默认当前活动 
- **强制包含的卡面ID**：同 “活动组卡”  

**示例：**
```
加成组卡 100 c[强制包含的卡面ID]
cn加成组卡 100 200 e112 mfy
```

### 21. **绑定信息 (绑定信息)**
-显示你在指定服务器绑定的账号和默认绑定，未指定则全部显示

```
[服务器]绑定信息
```

### 22. **快速换绑 (换绑/更新绑定)**
-快速更换默认绑定为指定编号的账号

```
[服务器]换绑|更换绑定[可选目标服务器] [编号]
```
**示例：**
```
tw换绑1 -> 将台服默认账号换绑到1号账号，不改变默认服务器
换绑cn1 -> 默认服务器设置为国服，并绑定国服下编号1的账号
```

### 23. **设置默认服务器 (设置默认服务器/更换默认服务器)**
-更换你所有指令的默认服务器

```
设置默认服务器|更换默认服务器[服务器]
```

### 24. **解除绑定 (/解除绑定)**
-解除指定编号对应账号的绑定关系（注意不会删除编号，不能解除默认绑定）

```
/[服务器]解除绑定 [编号]
```

### 25. **设置个人信息 (设置个人信息)**
-设置个人信息背景的高斯模糊级别和个人信息展示形态

```
[服务器]设置个人信息 [模糊+数字] [横屏|竖屏]
```
注： 高斯模糊0为关闭，`[竖屏]` 待开发，现在设置没用。
